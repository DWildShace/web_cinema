## Project Overview

Web-based movie ticket booking system. React frontend + ASP.NET Core 8 backend + PostgreSQL 16 in Docker on a Debian VM.

| Layer    | Technology                                    |
|----------|-----------------------------------------------|
| Frontend | React 18, Vite, Bun, Tailwind CSS v4, Zustand |
| Backend  | ASP.NET Core 8 Web API, MVC pattern           |
| ORM      | Entity Framework Core 8 (Code-First)          |
| Database | PostgreSQL 16 (Docker on Debian VM)           |
| Auth     | JWT Bearer tokens                             |
